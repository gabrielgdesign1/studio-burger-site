# 🍔 Studio Burger — Site Oficial

Site institucional da **Studio Burger**, uma das melhores hamburguerias gourmet de Goiânia — Setor Bueno.

## 🏆 Prêmios

- **Top 10 Melhores Hamburguerias do Brasil** — Burger Fest
- **Melhor Burger de Goiânia** — Prêmio Veja Comer & Beber (Cabral da Peste)

## ✨ O site

- Cardápio completo (138 itens em 9 categorias) com fotos, preços e descrições do cardápio digital oficial
- Busca instantânea e navegação por categorias
- Design dark gourmet com a paleta do logo (creme, verde e âmbar)
- Totalmente responsivo
- Links diretos para WhatsApp, iFood e Instagram

## 🛠 Stack

HTML + CSS + JavaScript puro — sem dependências, sem build. É só servir a pasta.

```bash
# rodar localmente
python -m http.server 8000
# → http://localhost:8000
```
